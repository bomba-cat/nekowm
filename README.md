# nekowm
A window manager with no special features, but at least it compiles fast

# Building
## Dependencies
- libxcb
- gcc (Change `Makefile` for a different compiler)
- make
## Compile
```bash
make
```
## Install
```bash
make install
```
## Uninstall
```bash
make uninstall
```
## Clean
```bash
make clean
```
